# Drug Classification System

## Project Overview

This project predicts the most appropriate drug for a patient using Machine Learning algorithms based on clinical parameters.

## Features

- Drug Recommendation System
- Interactive Streamlit Web Application
- Real-time Prediction
- Multiple Machine Learning Models
- Data Preprocessing and Feature Engineering

## Dataset

Drug200 Dataset

Features:
- Age
- Sex
- Blood Pressure
- Cholesterol
- Na_to_K Ratio

Target:
- Drug Type

## Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 85% |
| KNN | 87.5% |

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Joblib
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
